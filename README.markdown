# 学生考试成绩预测模型

## 项目背景
基于某教育平台提供的 63 万条学生匿名数据（含学习时长、出勤率、睡眠质量、学习方法等特征），构建回归模型预测学生期末考试成绩，并量化各因素对成绩的影响。

## 数据集
- 训练集：63 万条样本，12 个特征
- 测试集：27 万条样本
- 来源：Kaggle 公开数据集

## 项目结构
- `Predicting Student Test Scores.ipynb`：完整的数据分析、特征工程、建模与预测代码

## 方法
- 数据预处理：缺失值检查、有序变量映射、独热编码
- 特征工程：构造学习效率、睡眠强度、学习强度 3 个新特征
- 模型对比：线性回归、随机森林、LightGBM
- 超参数调优：对 LightGBM 进行学习率、树深、子采样等调优

## 结果
- 最优模型：LightGBM
- 验证集 R²：0.783
- MSE：77.25（相比线性回归降低约 2.2%）

## 运行方式
按顺序执行 notebook 中的所有单元格即可。

## 依赖库
- Python 3.12
- pandas, numpy, matplotlib, seaborn
- scikit-learn, lightgbm
